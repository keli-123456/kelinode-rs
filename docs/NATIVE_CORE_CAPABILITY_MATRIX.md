## Gate Semantics

- `RenderNative` is allowed without a preflight warning.
- `RenderNativeWithWarning` renders but produces a gray-preflight warning with the full capability context.
- `FallbackGo` is treated as a blocker in the Rust-native-only renderer.
- `Reject` is a blocker and prevents native config generation.

Unknown combinations are converted into an `Unsupported` reject entry so panel fields cannot be
silently dropped.
